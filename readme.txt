== Description ==

The plugin allows to export the names and email addresses of users, who have left comments on the blog. E-mail addresses of spammers will not be exported.

**Features**

* Check from which fields of the *wp_comments* table to export data (*comment_author*, *comment_author_email*).
* Select fields delimiter.
* **Show** or **Download** emails as file (and select file type: *.txt* or *.csv*).

**Support**

Support on [plugin page](http://ukraya.ru/comments-emails/support "Plugin Support").

== Installation ==

Upload the Comments Emails plugin to your blog, Activate it, go to: Tools - Comments Emails.
